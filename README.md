# vscode_extension
## 擴充套件
### vscode語言包
- Chinese (Traditional) Language Pack  
繁體中文套件
### Git版本控制
- Git Extension Pack  
包含以下套件  
-- Git History (git log)  
-- Project Manager  
-- GitLens  
-- gitignore   
-- Open in GitHub / Bitbucket / VisualStudio.com  
### Markdown
- Markdown All in One
  
- Markdown Preview Github Styling  
Markdown預覽Github的樣式

### 檔案預覽
- Path Intellisense  
智慧路徑提示，可以在你輸入檔案路徑時智慧提示

- PDF Viewer  
閱讀PDF的套件

- vscode-pdf

- Excel Viewer  
閱讀Excel、CSV的套件

### 前端套件  
- Live Server  
即時更新的本機伺服器，當網頁內容儲存馬上反應在瀏覽器的預覽頁面上

- HTML Snippets  
自動套出對應標籤

- Auto Close Tag  
自動閉合HTML/XML標籤

- Auto Rename Tag  
自動完成另一側標籤的同步修改

- indent-rainbow  
語法編輯時，使用Tab空格時顯示顏色，表語法層次性

- Code Runner
